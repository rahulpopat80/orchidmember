# Orchid Heights Society - Visitor Management System

ઓર્કિડ હાઇટ્સ સોસાયટી (૯૬ બ્લોક: A-101 થી B-1204) માટે બનાવેલી આ એક પ્રીમિયમ રિયલ-ટાઇમ મોબાઈલ વિઝિટર મેનેજમેન્ટ સિસ્ટમ છે.

આ રેપોઝીટરી ફ્રન્ટએન્ડ (Vercel પર હોસ્ટ કરવા) અને બેકએન્ડ (Render પર હોસ્ટ કરવા) બંને માટે તૈયાર કરવામાં આવી છે.

---

## 🚀 deployment Guide (કેવી રીતે લાઈવ કરવું?)

આ એપ્લિકેશનને ઇન્ટરનેટ પર એકદમ **ફ્રી (Free)** માં લાઈવ કરવા માટે નીચેના બે સ્ટેપ્સ ફોલો કરો:

### સ્ટેપ ૧: ફ્રન્ટએન્ડ લાઈવ કરો (Vercel પર)
તમે ઓલરેડી પ્રોજેક્ટ Vercel પર કનેક્ટ કર્યો છે. 
* ફ્રન્ટએન્ડ આપોઆપ ગિટહબ પુશ થતાં જ લાઈવ થઈ ગયું છે.
* આ લિંક પરથી સિક્યોરિટી અને રહેવાસીઓ પોતાના ફોનમાં લોગઇન કરી શકશે.

---

### સ્ટેપ ૨: બેકએન્ડ સર્વર લાઈવ કરો (Render.com પર)
રહેવાસીઓ અને સિક્યોરિટી ગાર્ડના અલગ-અલગ ફોન વચ્ચે રિયલ-ટાઇમ કનેક્શન (ઘંટડી વાગવી અને અપ્રૂવલ સિંક થવું) માટે ઓનલાઈન સર્વર જરૂરી છે. તે મફતમાં સેટ કરવા માટે:

1. **[Render.com](https://render.com/)** પર જાઓ અને તમારા GitHub એકાઉન્ટથી લોગઇન કરો.
2. **"New"** બટન પર ક્લિક કરી **"Web Service"** સિલેક્ટ કરો.
3. તમારા આ GitHub રિપોઝીટરી (`orchidmember`) ને સિલેક્ટ કરો અને કનેક્ટ કરો.
4. સેટિંગ્સમાં નીચે મુજબની વિગતો આપો:
   * **Name**: `orchidmember` (અથવા તમારી પસંદનું નામ)
   * **Language**: `Node`
   * **Branch**: `main`
   * **Build Command**: `npm install`
   * **Start Command**: `npm start`
   * **Instance Type**: `Free` (સંપૂર્ણ મફત)
5. **"Deploy Web Service"** પર ક્લિક કરો. ૨ મિનિટમાં તમારું સર્વર તૈયાર થઈ જશે.
6. Render ની ઉપર એક લિંક મળશે, જે આવી દેખાશે: `https://orchidmember-xxxx.onrender.com`

---

### સ્ટેપ ૩: સર્વર લિંક એપ્લિકેશનમાં સેટ કરો
સર્વર લિંક મળી ગયા પછી:
1. તમારા લાઈવ સોસાયટી પોર્ટલ પર **એડમિન પેનલ (Admin Panel)** ખોલો (પાસવર્ડ: `Admin123`).
2. ત્યાં છેલ્લા ટેબ **"⚙️ સેટિંગ્સ"** પર જાઓ.
3. તમારી **Render સર્વર લિંક** ત્યાં બોક્સમાં પેસ્ટ કરો અને **"Save"** બટન દબાવો.
4. બસ! હવે સોસાયટીના તમામ રહેવાસીઓ અને સિક્યોરિટીના ફોન આપોઆપ એકબીજા સાથે કનેક્ટ થઈ જશે અને મોટેથી ઘંટડી વાગશે!

---

## 🛠️ Local Testing (કમ્પ્યુટર પર ટેસ્ટ કરવું હોય તો)
જો તમારે ઈન્ટરનેટ વગર કમ્પ્યુટર પર જ ટેસ્ટ કરવું હોય, તો ગીગા બાઈટ ઈન્ટરનેટ કે લોકલ સર્વરની જરૂર નથી.
ફક્ત **`index.html`** ફાઇલ ડબલ-ક્લિક કરીને ખોલો. તેમાં સિક્યોરિટી ફોન અને ૩ સભ્યોના ફોન એકસાથે ટેસ્ટિંગ માટે સ્ક્રીન પર જ દેખાશે.
